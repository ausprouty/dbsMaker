# Exploration Smell Test

_Use this quick check before publishing any Exploration content._

A **smell test** is a fast instinct check to catch language that quietly drifts
from **inviting discovery** into **explaining or persuading**.

If something feels slightly uncomfortable to read aloud to a curious friend,
pause and run this test.
